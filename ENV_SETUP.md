# Environment Variables for OmniAgent

## Firebase Configuration (for Online DB sync)
Get these from Firebase Console: https://console.firebase.google.com

```
NEXT_PUBLIC_FIREBASE_API_KEY=
NEXT_PUBLIC_FIREBASE_AUTH_DOMAIN=
NEXT_PUBLIC_FIREBASE_PROJECT_ID=
NEXT_PUBLIC_FIREBASE_STORAGE_BUCKET=
NEXT_PUBLIC_FIREBASE_MESSAGING_SENDER_ID=
NEXT_PUBLIC_FIREBASE_APP_ID=
```

## Google Generative AI (required)
```
GOOGLE_GENERATIVE_AI_API_KEY=your_gemini_api_key
```

## Optional: Google Custom Search (for web search tool)
Get from: https://console.cloud.google.com/apis/api/customsearch.googleapis.com
```
GOOGLE_SEARCH_API_KEY=
GOOGLE_SEARCH_CX=
```

## Optional: OpenWeather API (for weather tool)
Get from: https://openweathermap.org/api
```
OPENWEATHER_API_KEY=
```
